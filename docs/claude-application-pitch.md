# Claude Pro/Max application pitch

## Project summary
`hmode` is a small open-source Python CLI for organising local AI workflows. It helps users manage model presets, reusable prompt templates, session notes, reminders, and usage-window timing so they can work more efficiently across different models and providers.

## Why it matters
People who use multiple AI tools often waste time switching settings, retyping prompts, and losing track of timing or context. `hmode` gives them one lightweight terminal tool to keep the workflow tidy.

## What it does
- save and switch model presets
- store reusable prompt templates
- record short session notes
- keep simple reminders
- plan early-morning primer timings for rolling usage windows
- export and import config for easy backup

## Why it is credible
- real working CLI
- documented usage
- automated tests
- CI workflow
- clean repository structure
- built with only the Python standard library

## One-line application angle
I built `hmode` as a public open-source tool to reduce friction in my daily AI workflow, and I want to keep improving it publicly with better docs, features, and community use.
